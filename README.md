# osk-framework 
osk-framework 是一个基于springboot 和 antd 的快速开发脚手架。目前是作为acl项目服务的web子项目进行开发，当完成acl部分后再考虑进行分离和开源计划。

还有一个计划是目前国内还没有基于云存储的网盘项目，想尝试第一个作为基于多家云服务商存储服务的网盘开源项目，也是打算基于此项目进行开发。
